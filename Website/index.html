Goals of the project

I am trying to create a groundwater flow simluation program. The scope of the project is not yet very defined, but I would like it to focus mainly around human and environmental health.
Being that I am sequestered at home during the pandemic, I figured that I had better try to make the best of this time. I am a fan of computers, and I like to challenge myself, and this project sure is a project. From only the basic things I've done so far, I've been challenged and frustrated more than I though possible. But that's really the way it should be.

Plans change. I will no longer be trying to create my own modelling program. It's much more important that I learn more than try to create something for a topic that I don't really understand that much.
